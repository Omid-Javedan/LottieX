# LottieX

A tiny, fast desktop Lottie player born out of pure frustration with having to upload animations to a website just to see what they look like

## Project Setup

```sh
bun install
```

### Compile and Hot-Reload for Development

```sh
bun dev
```

### Type-Check, Compile and Minify for Production

```sh
bun run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
bun lint
```
